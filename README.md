# Telco Customer Churn Prediction

## 📌 Project Overview
This project predicts whether a customer will churn using machine learning.

## 🚀 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Random Forest / XGBoost

## 📊 Dataset
Telco Customer Churn Dataset (IBM Sample Data)

## ⚙️ Features
- Data preprocessing
- Feature engineering
- Model training
- Evaluation (Accuracy, Precision, Recall, F1-score)

## 📈 Results
Achieved good accuracy and balanced performance on churn prediction.

## 💡 Use Case
Helps telecom companies retain customers by identifying potential churn.
